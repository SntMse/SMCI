SM人物情報管理番号™️（SMCI）はSMSBが管理するSM番号™️（SMN）の一種で人物の情報を正確に管理、識別する目的で用いられる番号である。同じSM番号であるSM管理番号（SMMN）などとは違い決して人物を管理する目的の番号ではない。SMCIは11桁で形成されており、初めの2桁にSM番号の統一記号「SM」、次の一桁に「P」がついているのが特徴である。2020年12月現在、管理されている人物の情報は350件を超える。SMCIを振る利点として、名前の不確かな人物、昔の友人の兄弟などうろ覚えの情報も登録でき、分かった際に更新することで、データロスを防ぐことができる。また、人物に関連する情報などを統括して管理することができ、SntMseの個人的なデータベースであるSMデータアーカイブ（SMDA）の管理などにも必須の仕組みだ。多種多様な情報を管理しており、各人物につき、メールアドレスや住所情報、Facebookのユーザー名まで60項目以上の情報を管理している。
